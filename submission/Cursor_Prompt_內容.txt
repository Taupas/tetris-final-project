我要做一個以 Python Flask 為前端、C 語言為後端的俄羅斯方塊遊戲，依據 PDF 規範進行設計。
請先提出整體架構（含 pointer、malloc/free、struct、資料結構、queue），再依照以下步驟逐一生成完整程式碼：

第一步：建立 C 核心引擎 c_engine/
● 用 struct 定義 Board（動態棋盤）、Piece（方塊）、Game（經典模式）、BlockGame（拼圖模式）
● 用 malloc / free 配置 Board.cells 一維陣列，以 row*cols+col 模擬二維棋盤（10×20 經典、8×8 拼圖）
● 用環形佇列 IntQueue（queue.c）實作 7-bag 隨機出塊（加分項目）
● 函式拆分：board.c、physics.c、tetromino.c、score.c、queue.c、engine_api.c、block_puzzle_api.c
● 編譯為共享函式庫 lib/tetris_engine.dll，匯出 engine_create、engine_tick、block_place 等 API

第二步：建立 Flask 後端 flask_app/app.py + services/engine_bridge.py
● 用 ctypes 載入 tetris_engine.dll（同進程呼叫，非 subprocess）
● 定義 EngineState、BlockEngineState 對應 C struct
● 提供以下 API（前綴 /tetris）：
  ○ GET /tetris：遊戲主頁面
  ○ POST /tetris/api/new：開新經典局
  ○ POST /tetris/api/input：玩家操作（left/right/down/rotate_cw/hard_drop）
  ○ POST /tetris/api/tick：重力下落
  ○ POST /tetris/api/block/new：開新拼圖局（8×8 拖放）
  ○ POST /tetris/api/block/place：放置托盤方塊
  ○ GET/POST /tetris/api/leaderboard、/tetris/api/puzzle/leaderboard：歷史分數

第三步：建立前端 templates/index.html + static/
● 主畫面兩按鈕：經典模式、拼圖模式（main.js 切換）
● game.js：Canvas 繪製 10×20，鍵盤操作，定時呼叫 /api/tick
● puzzle.js：8×8 拖放、幽靈預覽、方塊中心對齊與貼邊吸附（resolve_placement）
● 拼圖模式：隨機配色主題，全盤清除後換色；棋盤格線加強

第四步：建立 build.bat 與 run.bat
● build.bat：gcc -shared 編譯 c_engine/src/*.c → lib/tetris_engine.dll
● run.bat：自動編譯（若無 DLL）→ pip install -r requirements.txt → python flask_app/app.py
● 開啟瀏覽器：http://127.0.0.1:5000/tetris

第五步：建立 requirements.txt
● 僅需填入 flask>=3.0
